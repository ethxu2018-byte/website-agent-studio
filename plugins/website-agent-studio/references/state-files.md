# State Files

The plugin uses three core state artifacts:

## Project Profile

Defines:

- project identity
- codebase path
- audience
- business goal
- stack
- planning directories

## Project State

Defines:

- current phase
- active skill
- blockers
- next actions
- quality gate status

## Workflow Queue

Defines:

- the next concrete tasks
- their phase owner
- their priority
- their completion status

# Website Agent Studio Backend Integrations

Use this skill when the site needs real runtime behavior beyond static presentation.

## Read

- `../../references/state-files.md`
- `../../references/release-gates.md`

## Workflow

1. Identify the integration or backend capability.
2. Review env requirements first.
3. Keep validation, service logic, and route entry points separated.
4. Record any external blockers.
5. Update state and queue.

## Completion Criteria

- backend behavior is explicit
- env expectations are explicit
- preview/demo/live behavior is intentional if relevant

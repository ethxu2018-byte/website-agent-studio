# Website Agent Studio Handoff Continuity

Use this skill to make a website project transferable.

## Workflow

1. Identify the minimum handoff packet.
2. Write or refresh master instructions.
3. Write or refresh the rebuild spec.
4. Write or refresh acceptance and release context.
5. Update the project state so continuity is explicit.

## Completion Criteria

- another agent can safely continue the project
- approved project decisions are preserved in writing

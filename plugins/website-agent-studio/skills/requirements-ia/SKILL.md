# Website Agent Studio Requirements And IA

Use this skill to turn a brief into a real site structure.

## Read

- `../../references/phase-model.md`

## Workflow

1. Review the brief.
2. Define the sitemap.
3. Define user flows.
4. Define page-module structure.
5. Define priority requirements and acceptance conditions.
6. Update state and queue.

## Completion Criteria

- routes are defined
- user paths are defined
- implementation can proceed without inventing structure

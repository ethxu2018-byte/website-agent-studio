# Website Agent Studio Frontend Build

Use this skill when implementing the actual website UI.

## Workflow

1. Read requirements and visual rules first.
2. Build routes and sections modularly.
3. Keep content separate from rendering logic where possible.
4. Verify responsiveness and readability.
5. Verify interactions in a real browser.
6. Update state and queue.

## Completion Criteria

- routes render
- sections are modular
- the UI matches the approved design direction

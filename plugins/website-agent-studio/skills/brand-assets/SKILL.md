# Website Agent Studio Brand And Assets

Use this skill to keep the project visually coherent.

## Workflow

1. Review approved visual direction.
2. Confirm token or style rules.
3. Map assets to sections.
4. Align motion and copy with the product story.
5. Record any new design decisions.

## Completion Criteria

- section asset roles are clear
- visual rules are explicit
- engineering has a stable visual contract

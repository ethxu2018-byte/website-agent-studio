# Website Agent Studio QA And Release

Use this skill to verify the project honestly before release.

## Read

- `../../references/release-gates.md`

## Workflow

1. Run the quality gates.
2. Verify routes in a real browser.
3. Verify health, forms, and critical interactions.
4. Record what is production-ready, demo-ready only, and blocked.
5. Update state and queue.

## Completion Criteria

- checks actually ran
- browser verification actually ran
- the release status is documented

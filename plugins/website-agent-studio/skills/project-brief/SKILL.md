# Website Agent Studio Project Brief

Use this skill to define what the website is for.

## Workflow

1. Identify site type and audience.
2. Define the primary CTA.
3. Define secondary CTAs.
4. Write the positioning statement.
5. Write the business goal.
6. Update the project state.

## Completion Criteria

- audience is clear
- CTA is clear
- product intent is clear
